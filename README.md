# imdb_rating_scraper

This project automates the extraction of IMDb’s Top 250 movies using Python, demonstrating skills in web scraping, data parsing, and data cleaning.
It retrieves each movie’s title, release year, and IMDb rating, and exports the results into a structured CSV file for further analysis or visualization.

Tech Stack
Python 3.10+
Libraries: requests, beautifulsoup4, pandas, selenium (optional)
